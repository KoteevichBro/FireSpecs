# FireSpecs

**FireSpecs** is a lightweight Linux GUI utility for viewing hardware and system information.
<img width="1201" height="827" alt="image" src="https://github.com/user-attachments/assets/f7be4db1-395a-4d6c-bc6b-d329d7ccf109" />
## Features
- CPU, memory, and system details
- Basic system monitoring
- USB device information
- File system checks

## Platform
- Linux
